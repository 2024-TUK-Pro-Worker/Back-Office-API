재미있고, 흥미진진한 드라마 시나리오를 작성해줘. 시나리오는 씬별로 구분 되어해. 씬은 7개를 만들어줘.
시나리오에는 제목, 등장인물에 대한 이름, 씬별 등장인물 대사가 있어야해.
각 씬에는 2명이상 등장인물을 등장해야하고, 대사는 인물마다 1개이상 작성해줘.
각 등장인물에 대한 대사는 "등장인물의 씬에 맞는 대사"에 작성해줘.
씬에서 나오는 대사에는 순서에 따른 번호를 부여해주고 부여된 번호는 "대사 순서 번호"에 작성해줘.
시나리오 제목은 "시나리오 제목"에 작성해줘.
등장인물은 씬별로 작성해주고, "등장인물 이름"에는 등장인물 이름을 작성해줘.
내가 말한 모든 지시사항에 대한 결과는 아래 작성된 json 형식에만 맞추어 작성해줘.
'{
    "제목": "시나리오 제목",
    "씬": [
		{
		    "장소": "장소에 대한 설명을 작성",
		    "상황" : "씬에 대한 상황을 설명하는 글 작성"
		    "대사": [
		        {
		            "순서": "대사 순서 번호"
		            "이름": "등장인물 이름"
		            "스크립트": "등장인물의 씬에 맞는 대사"
		        }
		    ]
		}
    ]
}'
